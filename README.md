# ncs_base_script

Is a simple base script for ncs_core

## Installation:

Put in your cfg file:
```cfg
ensure ncs_base_script
```
and it's done.

## ncs_core:

ncs_base_sript is using [ncs_core](https://github.com/NextCitizens/ncs_core/).
